# Rest

One of 6 drives in the [[Drive System]].

**The desire to cease activity, be still.**

## Hormone Weights

- [[Serotonin]] +1.0 (contentment promotes rest)
- [[Endorphin]] +0.8 (euphoric stillness)
- [[Norepinephrine]] -0.7 (arousal opposes rest)

## When It's High

The agent wants to stop, be still, do nothing. Not withdrawal (which is fear-driven) — rest is contentment-driven.

## Related Emotions ([[Emotion Map]])

- High → "content", "calm", "lethargic"
