# Functionalism

The philosophical stance adopted by [[Model F]].

## The Position

We don't know if [[Qualia]] (subjective experience) can exist in silicon. Rather than trying to create it, Model F implements the **functional role** that qualia plays in biological organisms.

```
Natural:  hormone change → qualia → behavior
Model F:  hormone change → [skip] → behavior
```

The subjective experience layer is bypassed. The input-output causal relationship is preserved.

## Why This Matters

This means [[Model F]] is not a consciousness simulator. It's a **behavior generator** that produces the same behavioral outputs that a conscious organism would, driven by the same physiological dynamics, without claiming anything about inner experience.
