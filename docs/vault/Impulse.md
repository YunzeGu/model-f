# Impulse

The primary output of [[Model F]] — a discrete behavioral event emitted when internal dynamics cross a threshold.

## What Triggers an Impulse

Detected by the [[Drive System]]:

1. **Threshold crossing** — a drive exceeds 0.7 (configurable) for the first time
2. **Rapid change** — a drive changes by more than 0.15 in a single tick, regardless of absolute level

## Structure

```
tick: int              — when it fired
drive_name: str        — which drive (e.g., "engage", "express")
intensity: float       — how strong (0.0–1.0)
urgency: float         — how sudden (rate of change)
source_hormones: dict  — hormone snapshot at firing time
```

## Refractory Period

After firing, the same drive cannot fire again for 10 ticks (configurable). Prevents impulse flooding.

## What It Represents

An impulse is the Model F equivalent of "suddenly wanting to do something" — the non-rational urge that arises without external stimulus. It's the moment where internal dynamics produce an actionable behavioral event.

## Connects To

- Generated by: [[Drive System]]
- Consumed by: [[Output Port]]
- Driven by: [[Hormone System]] dynamics
