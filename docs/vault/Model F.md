# Model F

A **bionic subconscious** — a [[Floating Brain]] that generates non-purposive [[Impulse|impulses]] from internal [[Hormone System|hormone dynamics]].

## Core Pipeline

```
[[Hormone System]] → [[Drive System]] → [[Impulse]] → [[Output Port]]
```

## Design Stance

- [[Functionalism]] — bypass [[Qualia]], implement its causal role directly
- [[Non-Purposive Autonomy]] — machines that *want to act* without a reason
- [[Pluggable Architecture]] — capabilities attach later via [[Input Port]] / [[Output Port]]

## Components

- [[Hormone System]] — 6 hormones with natural dynamics
- [[Drive System]] — 6 behavioral drive dimensions
- [[Emotion Map]] — read-only labeling layer
- [[Engine]] — tick-based orchestrator
