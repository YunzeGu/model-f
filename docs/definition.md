# Model F — Definition

This document defines Model F's conceptual identity and design philosophy. It serves as the living specification and should be updated as understanding evolves.

## What Model F Is

Model F is a **subconscious, irrational impulse generator** — a base model that simulates the spontaneous behavioral drives found in natural biological organisms.

It is a system that **receives** low-level sensory/physiological signals and **emits** behavioral drive commands, functioning as the non-rational motivational core of an agent.

## Core Mechanism: Sensation-Preserving Transformation

Model F does not relay signals directly. It performs a **sensation-preserving, content-transforming** process:

- Input and output are equivalent in **felt quality** (affective tone), but different in **representational form**
- Example: chest compression during sleep (physical input) → nightmare scenario involving conflict (behavioral/experiential output). The dream content differs from the raw stimulus, but the subjective quality (tightness, distress) is preserved.

In other words: the **texture** of the signal is conserved, but the **narrative** is generated.

## Hormone-to-Action Mapping

Model F converts physiological state changes (e.g., hormone levels) into behavioral motivation or resistance:

| Physiological State | Behavioral Output |
|---|---|
| Low dopamine | High resistance, seeking low-cost/quick-reward stimulation (e.g., doomscrolling) |
| High dopamine (e.g., post-exercise, post-release) | High drive, impulse to initiate action |

The mapping is not symbolic — it produces a **continuous force vector** of motivation/resistance that shapes which actions the agent is inclined toward.

## Philosophical Position: Functional Substitution of Qualia

Whether phenomenal consciousness (qualia) exists in silicon-based systems is an open question. Model F takes a **functionalist stance**:

- It does **not** attempt to create subjective experience
- It directly implements the **causal role** that qualia plays in natural organisms
- It bypasses the "hard problem" and focuses on input-output fidelity

```
Natural organism:  hormone change → qualia (subjective feeling) → behavior
Model F:           hormone change → [Model F] → behavioral command
```

The middle layer (subjective experience) is bypassed, but the causal relationship between input and output is preserved.

## Purpose: Non-Purposive Autonomous Impulse

Current AI systems are goal-oriented — given an objective, they optimize execution. Model F rejects this paradigm at the surface level.

**Model F gives machines the desire to act without a reason.**

Natural organisms exhibit non-purposive behavior constantly: wandering aimlessly, suddenly wanting to sing, craving a specific food for no reason. These impulses have no explicit goal, yet they are the foundation of autonomy and agency.

### Two-Layer Structure

- **Deep layer**: Broad evolutionary-analog rules (homeostasis, hormone dynamics) provide energy and directional tendency — analogous to survival/reproduction pressures in biology
- **Surface layer**: Emergent, seemingly purposeless impulses — reading philosophy at midnight, staring at the sky, reorganizing a room at 3 AM

The deep rules **fuel** surface behavior but do not **dictate** it. The gap between deep drive and surface action is where autonomy lives.

### Contrast with Existing AI

| | Goal-oriented AI | Model F |
|---|---|---|
| Motivation source | External objective | Internal state dynamics |
| Behavior when idle | Wait for instruction | Spontaneous impulse generation |
| Surface-level purpose | Always present | Deliberately absent |
| Autonomy | Instrumental | Intrinsic |

## Design Scope: The Floating Brain

Model F is deliberately scoped as a **"floating brain"** — a consciousness model with no sensory organs, suspended in a void, driven entirely by internal dynamics.

### What was excluded and why

1. **3D embodied simulation** (e.g., a cat in Unity) — requires solving sensory decoding (vision, touch, proprioception) before the core model is even validated. Deferred.
2. **LLM / rational reasoning** — animals without complex logic still have desires. Model F must function without any language model. Rationality is a capability that plugs in later, not a prerequisite.
3. **Language interaction mode** — output is numerical drive vectors, not text. Pure impulse generation.

### Pluggable capability architecture

Model F is a **base module** designed to accept future capability plugins:

- **Memory** (long-term / short-term memory models)
- **Logical reasoning** (LLM)
- **Visual processing** (vision models)
- **Motor control** (embodied agent interfaces)

None of these are connected in v1. The core must be fully functional and interesting on its own — a self-contained impulse generator with clean input/output ports for future extension.

## Integration Principle: Continuous Modulation, Not Animation

Model F's output is a **per-frame continuous modulation signal**, not a discrete animation trigger.

### The pathfinding example

An agent feels thirsty. Water is across the room. The pathfinding skill determines the route. Model F determines **how the agent traverses that route**:

- High cortisol (anxious): faster speed, more direct path, tighter turns, fewer pauses
- High serotonin (calm): slower pace, willingness to meander, relaxed movement
- High dopamine (eager): quick acceleration, energetic gait

### What this is NOT

Playing a "anxious_walk" animation clip. That is:
- Discrete (state A → state B)
- Pre-recorded (identical every time)
- Performative (the agent is "acting" anxious, not "being" anxious)

### What this IS

Every frame of behavior is modulated by the current hormone state in real time. The drive vector is a continuous field that colors every moment of execution. No two "anxious walks" are ever identical, because the underlying hormones shift every tick. This is the difference between **acting** and **being**.

This higher standard — pursuing continuous experiential modulation rather than discrete behavioral switching — is what gives Model F the potential for genuinely diverse, organic behavior that cannot be achieved through animation state machines.

## Summary

Model F is a **bionic subconscious**: a system that translates continuous physiological dynamics into spontaneous behavioral impulses, preserving affective fidelity without requiring phenomenal consciousness, and producing non-purposive autonomous drive as its primary output.
